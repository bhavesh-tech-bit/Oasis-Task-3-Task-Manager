# Oasis-Task-3-Task-Manager
This repository contains the projects I completed during my Web Development Internship at OASIS Infobyte. The tasks include a To-Do List App  built using HTML, CSS, and JavaScript.
Just built my own Task Manager from scratch using HTML, CSS & JavaScript!
Here's what I built and what I learned:

What it does:
•Add, edit, and delete tasks in real time
•Mark tasks as complete - they move automatically to a "Completed" section
•Clean two-column layout separating Pending vs Completed tasks

 Tech used:
•HTML - structured the app layout with semantic elements
•CSS - styled with a gradient background, card UI, responsive flexbox layout, and color-coded action buttons
•JavaScript - handled all DOM manipulation: dynamically creating task elements, managing button events (complete , edit , delete ), and moving tasks between lists

 Key concepts I practiced:
•Dynamic creation with createElement and appendChild
•Event handling with onclick functions
•Separation of concerns across HTML / CSS / JS files
•Building UI components (like button groups) programmatically

This was a great exercise in understanding how the works without relying on any frameworks. No React, no libraries - just pure fundamentals!
